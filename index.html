Making a pizza is similar to coding. There are certain things we have to do first. Like making the base, letting it rise, mould the base out, add the source, add the toppings, and the cheese, bake, and cut and serve. Not all pizzas end up the same, but they do start out the same way as long as you follow the order corectly. You can have everything you need to make many different pizzas using the same base structure that is set up before hand.

This is how you make a pizza

There are some specific steps you need to take to make a pizza, we will list them here

  1. make the pizza base
  2. add the tomato sauce
  3. add the topping
  4. add the cheese
  5. bake
  6. cut
  7. serve
the toppings are different, they change for every occasion and taste. there is no order to toppings. So we will list them like this.
  • Pepperoni
